# Tulipanes bajo otra luna

Mini sitio romántico estático listo para GitHub Pages.

## Archivos

- `index.html`: entrada principal para GitHub Pages
- `tulipanes-bajo-otra-luna.css`: estilos
- `tulipanes-bajo-otra-luna.js`: lógica y textos editables en `CONFIG`
- `tema.mp3`: música de fondo local

## Publicar en GitHub

1. Sube este contenido a un repositorio en GitHub.
2. En GitHub ve a `Settings` > `Pages`.
3. En `Build and deployment`, elige `Deploy from a branch`.
4. Selecciona la rama `main` y la carpeta `/ (root)`.
5. Guarda y espera a que GitHub publique el sitio.

## Editar contenido

En `tulipanes-bajo-otra-luna.js`, dentro de `CONFIG`, puedes cambiar:

- `startDate`
- `photoUrl`
- `musicSrc`
- `musicVolume`
- `letterTitle`
- `paragraphs`

## Nota

`tema.mp3` ya está configurado como música de fondo. Si cambias el nombre del archivo, actualiza `musicSrc`.
